# Le Village NIRD — NomÉquipe

Mini-site interactif réalisé pour la Nuit de l'Info 2025 — Sujet : "Le Village Numérique Résistant".

## URL publique
https://<votre-nom-utilisateur-github>.github.io/<nom-du-repo>  (sera active après publication sur GitHub Pages)

## Description
Parcours ludique pour sensibiliser élèves, enseignants et familles à la démarche NIRD (Numérique Inclusif, Responsable et Durable). Le site propose 5 "maisons" thématiques, des quiz, et un système de badges.

## Technologies
- HTML, CSS, JavaScript (Vanilla)
- Hébergé sur GitHub Pages

## Comment tester localement
1. Ouvrir `index.html` directement dans le navigateur
   - Ou, pour un serveur local : `python -m http.server 8000`
   - Puis visiter `http://localhost:8000`

## Déploiement (GitHub Pages)
1. Pousser le repo sur GitHub (voir instructions ci-dessous).
2. Activer GitHub Pages : Settings → Pages → Branch = `main`, Folder = `/ (root)`.
3. L'URL sera : `https://<votre-utilisateur>.github.io/<nom-du-repo>/`
